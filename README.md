🚀 A modern and responsive Goal Tracker web application built with React and Vite. The app allows users to create goals, track progress, earn XP, maintain streaks, and manage goals through a clean dashboard interface. It supports both English and Persian languages with automatic RTL/LTR layout switching.



## Features Checklist

### Core Features

*  Dashboard page
*  Goals list page
*  Create new goal page
*  Goal details page
*  Categories overview page
*  Settings page
*  404 Not Found page

### Goal Management

*  Create goals
*  Edit goals
*  Delete goals
*  Pause and resume goals
*  Mark goals as completed
*  Search goals
*  Filter goals
*  Sort goals
*  Category goals

### Progress Tracking

*  Progress logs
*  Automatic progress percentage calculation
*  Auto-complete goals when target is reached
*  Completed goals archive

### Gamification

*  XP system
*  Streak system
*  Dashboard statistics

### UI Features

*  Responsive design
*  Dark mode
*  Light mode
*  Progress bars
*  Cards and icons
*  Empty states

### Internationalization

*  English language
*  Persian language
*  RTL support
*  LTR support

### Data Persistence

*  LocalStorage support

## How to Run

### Clone the repository

git clone <repository-url>

### Go to Folder

cd .\goal-tracker\ 


### Install dependencies

npm install 

### Start development 

npm run dev




## Routes / Pages

| Route       | Description         |
| ----------- | ------------------- |
|             |                     |
| /goals      | Goals list page     |
| /goals/new  | Create goal page    |
| /goals/     | Goal details page   |
| /categories | Categories overview |
| /settings   | Settings page       |
| *           | 404 page            |



## Language & RTL/LTR Support

The application supports two languages:

* English (LTR)
* Persian (RTL)

When the language changes, the entire layout direction changes automatically to provide a better user experience.



## XP Rules

* Every progress entry gives +20 XP.
* XP is accumulated and displayed on the dashboard.



## Streak Rules

* Logging progress on consecutive days increases the streak.
* Missing a day resets the streak.



## Responsive Design

The application is fully responsive and optimized for:

* Desktop devices
* Tablets
* Mobile devices
